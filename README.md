# English Coach B1

Projeto simples em Python para praticar inglês no nível B1.

Esta versão não usa IA real. Ela usa regras simples para ajudar na revisão de frases, vocabulário e prática de conversa.

## O que o projeto faz

- Corrige algumas expressões informais
- Remove espaços extras
- Ajusta letra maiúscula no começo da frase
- Adiciona ponto final quando necessário
- Gera perguntas de conversa
- Mostra palavras úteis em inglês

## Conceitos de Python praticados

- Variáveis
- Listas
- Dicionários
- Funções
- `if`, `elif` e `else`
- `while`
- `random.choice`
- Manipulação de texto com `strip`, `replace`, `split` e `join`

## Como executar

No Google Colab ou em um ambiente Python, rode o arquivo:

python english_coach_b1.py
